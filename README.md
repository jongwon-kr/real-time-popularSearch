# 실시간 인기 검색어 프로그램
## 프로젝트 설명
네이버뉴스의 인기, 연예 카테고리에 있는 데이터들을 분석하여 많은 관심을 가진 검색어를 보여주기 위해 제작 되었다.

## 서비스 내용
- 인기 카테고리 인기 검색어 top 10
- 연예 카테고리 인기 검색어 top 10

## 프로젝트 기간
2022.07 ~ 2022.08

## 개발 환경 및 언어
- Eclipse
- Java

## 주 활용 클래스
- Swing
- URI
- URL

## 활용 데이터
